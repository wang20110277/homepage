## API 接口列表

### 1. 声纹对比

**接口地址：** `POST /api/compare`

**功能说明：** 上传两个音频文件，提取声纹向量并计算相似度

#### 请求参数

| 参数名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| audio1 | File | 是 | 第一个音频文件（.wav 或 .mp3） |
| audio2 | File | 是 | 第二个音频文件（.wav 或 .mp3） |

**请求方式：** `multipart/form-data`

**文件限制：**
- 支持格式：`.wav`、`.mp3`
- 文件大小：最大 50MB

#### 请求示例

**cURL:**
```bash
curl -X POST http://10.162.5.210:7862/api/compare \
  -F "audio1=@/path/to/audio1.wav" \
  -F "audio2=@/path/to/audio2.wav"
```

**Python (requests):**
```python
import requests

url = "http://10.162.5.210:7862/api/compare"
files = {
    'audio1': open('audio1.wav', 'rb'),
    'audio2': open('audio2.wav', 'rb')
}
response = requests.post(url, files=files)
print(response.json())
```

**JavaScript (fetch):**
```javascript
const formData = new FormData();
formData.append('audio1', audio1File);
formData.append('audio2', audio2File);

fetch('http://10.162.5.210:7862/api/compare', {
  method: 'POST',
  body: formData
})
.then(res => res.json())
.then(data => console.log(data));
```

#### 响应结果

**成功响应 (200 OK):**
```json
{
  "success": true,
  "similarity": 0.8567,
  "similarity_percent": 85.67,
  "message": "两个音频的余弦相似度为: 0.8567"
}
```

**字段说明：**
| 字段 | 类型 | 说明 |
|------|------|------|
| success | boolean | 请求是否成功 |
| similarity | float | 余弦相似度（0-1） |
| similarity_percent | float | 相似度百分比（0-100） |
| message | string | 结果描述 |

**相似度参考值：**
| 相似度范围 | 判断 |
|-----------|------|
| 0.85 - 1.00 | 高度相似，极可能是同一人 |
| 0.70 - 0.85 | 相似，可能是同一人 |
| 0.50 - 0.70 | 一般相似，需要进一步确认 |
| 0.00 - 0.50 | 不相似，不太可能是同一人 |

**错误响应示例：**

| 错误码 | 说明 |
|--------|------|
| 400 | 缺少文件/文件名为空/格式不支持 |
| 400 | 有效语音长度不足（最少需要3秒） |
| 500 | 服务器内部错误 |

```json
{
  "error": "请上传两个音频文件"
}
```

```json
{
  "error": "只支持 .wav 和 .mp3 格式"
}
```

```json
{
  "error": "声纹提取失败: 有效语音长度不足 3.0 秒"
}
```

### 2. 健康检查

**接口地址：** `GET /api/health`

**功能说明：** 检查服务状态和模型加载情况

#### 请求参数

无

#### 请求示例

**cURL:**
```bash
curl http://10.162.5.210:7862/api/health
```

#### 响应结果 (200 OK)

```json
{
  "status": "ok",
  "model_loaded": true,
  "model_path": "/path/to/model.onnx"
}
```

**字段说明：**
| 字段 | 类型 | 说明 |
|------|------|------|
| status | string | 服务状态 |
| model_loaded | boolean | 声纹模型是否已加载 |
| model_path | string | 模型文件路径 |

## 音频处理参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 采样率 | 16kHz | 音频自动重采样到16kHz |
| 有效时长 | 3-20秒 | 最少3秒，最多取前20秒 |
| 频率范围 | 200-3500Hz | 带通滤波处理 |
| 声纹维度 | 192 | CAM++ 模型输出 |
